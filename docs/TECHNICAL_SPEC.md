# 英语视频生产环节：技术说明

这个项目把视频制作分成了几个关键步骤，每个步骤由一个专门的脚本（Python文件）来完成。

---

## 1. 整理文案 (`step1_analyze.py`)
*   **干什么**：电脑先看你的文章，把太长的句子切成比较短的小块，这样声音和画面才好配。
*   **作用**：它还会分析出文案里有哪些重点词，用来作为下一步找视频的“搜索词”。

## 2. 合成配音 (`step2_tts.py`)
*   **干什么**：把整理好的文案喂给 VibeVoice 语音模型，生成好听、自然的朗读声音。
*   **作用**：每一句话都会生成一个声音文件，并记录下这段声音到底有几秒几毫秒长。

## 3. 配上画面 (`step3_video.py`)
*   **干什么**：根据文案里的重点词，从网上自动下载匹配的视频素材。
*   **作用**：如果音频很长，会自动下载多个镜头，保证画面一直在动，不会死板。

## 4. 制作字幕 (`step5_subtitle.py`)
*   **干什么**：用高精度的声音分析工具，算出朗读音频里每一个字出现的准确时刻。
*   **作用**：如果一句话太长，它会自动把字幕变短，分多行显示，保证屏幕好看、不挤。

## 5. 合并渲染 (`step7_merge.py`)
*   **干什么**：像胶水一样，把生好的声音、下好的视频画面、还有字幕文件全部融合在一起，导出一个最终的 MP4 视频。
*   **作用**：它还会自动压低背景音乐，让人的朗读声音更清晰，效果就跟专业剪辑师剪出来的一样。

## 6. 自动投稿 (`step9_upload.py`)
*   **干什么**：把做好的视频、封面、标题自动发到你的 B 站账号里。
*   **作用**：里面带了特殊的“浏览器伪装”代码，能有效防止 B 站把它当成机器人操作。
