## 📊 项目全面分析

### 项目本质
这是一个**自动化英文学习视频生成系统**：从markdown文章 → AI生成语音 → 匹配免费视频 → 自动合成带双字幕的B站视频。

### 核心工作流程（8个步骤）
```
输入文章 → 语义分析(DeepSeek) → TTS语音合成(VibeVoice) → 视频搜索下载(Pexels/Pixabay) 
         → 字幕对齐(WhisperX) → 翻译+ASS特效字幕 → FFmpeg合成 → B站上传
```

### 技术亮点
| 特性 | 说明 |
|------|------|
| **2025-2026前沿技术** | DeepSeek V3, VibeVoice TTS, WhisperX, Kolors封面生成 |
| **GPU优化** | TTS批量处理、NVENC硬件加速、显存监控 |
| **稳定性设计** | API限流、缓存系统、下载备用URL、视频去重 |
| **完整自动化** | 端到端无需人工干预 |
