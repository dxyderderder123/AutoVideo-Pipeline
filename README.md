## ⚙️ 项目架构与工作流程

本项目实现了从**输入文章到视频成品**的全链路自动化逻辑。系统具备硬件感知（GPU锁）、API熔断降级和多线程并行能力，确保了生产环境的稳健性。

### 1. 核心自动化管线 (Pipeline)

系统将视频制作逻辑拆解为 9 个独立步骤，并由核心逻辑统筹调度：

| 阶段 | 功能描述 | 核心技术 |
| :--- | :--- | :--- |
| **01 分析** | 语义切片与关键词提取 | LLM (NLP 语境分析) |
| **02 配音** | 高拟真语音合成 | **VibeVoice** 推理引擎 |
| **03 素材** | 视觉素材抓取与熔断保护 | Pexels/Pixabay API 接入 |
| **04 对齐** | 音频与文本毫秒级强制对齐 | WhisperX / 词频比例算法 |
| **05 字幕** | 视觉短句边界切分 | 动态文本折行逻辑 |
| **06 翻译** | 双语字幕生成与渲染 | LLM 翻译 + libass 渲染 |
| **07 混剪** | 视音频底层物理合成 | **FFmpeg** 媒体处理滤镜链 |
| **08 封面** | AI 艺术封面生成 | 扩散模型 (Diffusion Model) |
| **09 发布** | 自动化分发与上传 | 协议指纹模拟 (WAF 对抗) |

### 2. 批量生产能力

本项目支持**一键扫描式批量生产**：

- **自动化入口**：系统自动扫描 `workspace/input/` 目录下的所有 Markdown 文稿。
- **并行调度**：自动检测 GPU 显存负载，动态决定并行任务数，在保证硬件安全的前提下最大化生产效率。
- **一键运行**：通过 `scripts/run_batch.bat` 即可启动完整的流水线任务。
